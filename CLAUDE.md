This app is a fantasy baseball-like application with custom scoring intended to reward participants who choose the worst players. Claude should not add features without permission and should be careful not to create security vulnerabilities. The app should remain as simple to deploy as possible and should be cheap to run.

The scoring is based on aggregating daily stat lines of players. the basic unit of data is a statline. each day, the app will record the statline for every player, as well as the team that owns that player. this way, the stat lines can be aggregated into player-season totals as well as team-season totals and player-team-season totals.
